# Streamlit Password Game

Este projeto é um jogo de desafios por fases usando Streamlit. Agora, as senhas corretas podem ser salvas em um serviço de armazenamento de senhas (como pass) ou na nuvem, diretamente pelo dashboard.

## Como rodar

1. Instale as dependências:
   ```bash
   pip install streamlit
   ```
2. Execute o app:
   ```bash
   streamlit run main.py
   ```

## Funcionalidades
- Desafios por fases
- Senhas corretas podem ser salvas em um serviço externo (em breve)
